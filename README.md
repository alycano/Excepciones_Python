# Proyecto de Excepciones Python - SENA

El manejo de excepciones en Python es una técnica fundamental que nos permite controlar los errores que surgen durante la ejecución de un programa, evitando que este se detenga de manera abrupta y afecte la experiencia del usuario. Su importancia radica en la creación de software robusto y profesional, ya que en el mundo real, especialmente en el desarrollo backend, debemos anticiparnos a situaciones inesperadas como ingresos de datos incorrectos, fallos en conexiones de red o archivos inexistentes. En este repositorio, implementé estos conceptos mediante el uso de la estructura try-except-else-finally. En los ejercicios prácticos, utilicé el bloque try para encapsular las operaciones sensibles, como divisiones matemáticas y manipulación de archivos, mientras que con los bloques except específicos capturé errores de tipo ValueError y ZeroDivisionError. Además, apliqué el bloque else para confirmar ejecuciones exitosas y el finally para asegurar el cierre de recursos, como en el caso del manejo de archivos. Finalmente, llevé el aprendizaje un paso más allá creando excepciones personalizadas que permiten manejar reglas de negocio específicas, demostrando cómo estas herramientas se pueden integrar en proyectos reales de gestión de inventarios o sistemas de información como los que trabajamos en el programa ADSO.
